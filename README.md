# RDBR-task-front

## Installation
Define api url in public/config.json file, like: "http://localhost:8000/api/"
```
git clone https://github.com/MaestroError/RDBR-task-front.git
cd RDBR-task-front
npm install
npm run serve
// OR
npm run build
```
